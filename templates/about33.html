About This Project

This project is a Bangla Character Recognition system using Optical Character Recognition (OCR). It detects and identifies Bangla characters from images using a Convolutional Neural Network (CNN) model.

How It Works

  • Upload an image containing Bangla text.
  • The model processes the image to extract characters.
  • The predicted text is displayed along with an annotated image.
